Snow Programming Language
============================

Snow Programming Language will solve your problem of where to start with documentation,
by providing a basic explanation of how to do it easily.

Look how easy it is to use:

    import Snow Programming Language
    # Get your stuff done
    Snow Programming Language.do_stuff()
